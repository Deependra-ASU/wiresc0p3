## File Analyzer

This tool is used to scan C files to find Vulnerablilites in them.

## Usage

To use this tool, Run checkfile.py <Filename or Folder name>

Example:

`python3 checkfile.py filename`

`python3 checkfile.py /path/to/dir/to/scan`
